Class Admins_Model

Admins_Model Class

Handles the database actions for admin users management.

Data Structure: 'fist_name' 'last_name' (required) 'email' (required) 'mobile_number' 'phone_number' (required) 'address' 'city' 'state' 'zip_code' 'notes' 'id_roles' 'settings' >>> array that contains user settings (username, password etc)
